# Regular_Expression_Java
Javaで正規表現を扱う

## 実行

``` sh
mvn clean compile exec:java -Dexec.mainClass="sample.regularexpression." -Dexec.args="'${PWD}/src/main/resources/text.txt' '${PWD}/src/main/resources/pattern.txt'"
```
